Reading: #WordJournal

Word: Consistency

This reading can be summarized by consistency because it talks mainly about the elements of great style guides and content strategy, which revolves around your user interface build consistent in its element styles. Figma, before having their design system, mentioned that they had a lack of consistency, which led them to create their component library. The spatial system of a website should have set rules for how the interface elements are measured, sized, and spaced. Typography should be the same, making sure that you keep in mind how many weights or variations you'll need for clear hierarchies. Font sizes and line heights should also be similar throughout a site, establishing a base font size and line height 1.5x the font size. The site should use common vocabulary and microcopies to ensure consistency as well. Overall, a defined and consistent style guide is essential for developing a coherent product. This is important for me to remember in my career because I will be developing web applications using companies' UI libraries and component trees, so knowing why these are significant and when to use them is essential.
